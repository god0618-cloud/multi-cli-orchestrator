# Redaction Checklist

Before any prototype code or example is copied into this repository, verify:

- No absolute private paths such as `/Users/<name>/...`.
- No internal business names, store names, employee names, contracts, or private process data.
- No tokens, cookies, API keys, bearer tokens, MCP credentials, SSH keys, or OAuth artifacts.
- No private screenshots unless anonymized.
- No native CLI memory, soul, profile, or stable knowledge-base content.
- No task IDs tied to private work unless converted into sanitized fixtures.
- No internal URLs, database hosts, or company-specific domains.
