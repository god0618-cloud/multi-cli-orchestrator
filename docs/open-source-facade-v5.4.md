# Open Source Facade v5.4

v5.4 focuses on the repository storefront: what a new visitor sees before they decide to clone, star, or contribute.

## Changes

- README now shows a dashboard screenshot near the top.
- README includes a visual walkthrough table for dashboard, replay, and adapter matrix.
- Launch playbook is updated from old v3 public MVP wording to the current v5 public Alpha posture.
- Messaging explicitly avoids unsafe claims such as arbitrary shell execution or real concurrent provider execution.
- GitHub topics should include AI coding, Agent OS, workflow automation, and local-first orchestration terms.
